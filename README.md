## How to Run

In the project directory, you can :

1) Install dependencies :
```
yarn
```

2) Run this app on your browser :
```
yarn start
```

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in a browser.
